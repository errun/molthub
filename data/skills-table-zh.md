# Skills（中文版）

| Skill | 定位（通俗版） | 适合谁 / 解决什么问题 | 安装命令 |
|---|---|---|---|
| openclaw-feeds | 把新闻/财经/游戏等 RSS 聚合成“统一数据流”，方便做资讯雷达/日报周报。 | 想做“每日资讯/爆款雷达”的人；解决“信息源太散、不好抓”的问题。 | `clawhub install openclaw-feeds` |
| rssaurus | 如果你用 RSSaurus 读 RSS：用命令行批量“列未读/标已读/收藏/批处理”。 | 重度 RSS 用户；解决“手动点太慢、想批量管理订阅”的问题。 | `clawhub install rssaurus` |
| vercel | 用命令行管理 Vercel：部署、看日志、改环境变量、列项目等。 | 做网站/前端发布的人；解决“发布流程重复、想自动化部署”的问题。 | `clawhub install vercel` |
| wordpress | 用 WordPress API 自动化：列文章、发草稿、改页面、管分类标签等。 | 内容站运营/自媒体团队；解决“发文、改站点太机械重复”的问题。 | `clawhub install wordpress` |
| wpclaw-lite | 面向 WooCommerce：通过连接器拉订单、查商品，做基础运营自动化。 | 做跨境/独立站电商的人；解决“订单/商品数据想自动拉取处理”的问题。 | `clawhub install wpclaw-lite` |
| pinch-to-post | 更全能的 WordPress 自动化：发文章、媒体、SEO、批量操作等。 | 追求“内容生产流水线”的人；解决“内容发布+SEO+素材处理分散”的问题。 | `clawhub install pinch-to-post` |
| postiz | 多平台内容排程：一套内容分发到多渠道、定时发布、管理队列。 | 做增长/运营分发的人；解决“多平台重复发、定时发布难管理”的问题。 | `clawhub install postiz` |
| news-summary | 把新闻抓下来并生成“可读摘要/要点”，用来做每日简报。 | 想省时间看新闻的人；解决“信息太多、想直接看重点”的问题。 | `clawhub install news-summary` |
| blogwatcher | 监控网站更新：订阅博客/RSS，自动发现“谁更新了”。 | 盯竞品/行业动态的人；解决“每天手动刷网站很浪费时间”的问题。 | `clawhub install blogwatcher` |
| clawops | “技能编排/调度”：把多个 skill 组织起来，串成自动化工作流。 | 有多套自动化需求的人；解决“技能多但不会串联/缺统一调度”的问题。 | `clawhub install clawops` |
| openclaw-anything | OpenClaw 万能工具箱：安装、诊断、连渠道、管任务等。 | OpenClaw 管理员/折腾党；解决“OpenClaw 运维命令太散、不好记”的问题。 | `clawhub install openclaw-anything` |
| openclaw-server-secure-skill | 服务器安全加固清单/脚本：把运行 OpenClaw 的服务器做基础防护。 | 要上生产/对安全敏感的人；解决“服务器裸奔、怕被扫被入侵”的问题。 | `clawhub install openclaw-server-secure-skill` |
| rss-ai-reader | 自动读 RSS → 大模型生成中文摘要 → 推送到 Telegram/飞书/邮件。 | 想做“订阅源=内容产品”的人；解决“订阅太多看不过来、需要自动总结推送”的问题。 | `clawhub install rss-ai-reader` |
