# Skills (English)

| Skill | What it does (plain English) | Best for / Problem it solves | Install command |
|---|---|---|---|
| openclaw-feeds | Aggregates RSS from news/finance/gaming into one clean data feed for reports and “trend radar”. | Anyone building a daily briefing or “what’s hot” dashboard; solves “sources are scattered and hard to collect”. | `clawhub install openclaw-feeds` |
| rssaurus | If you use RSSaurus: bulk manage feeds from the terminal (list unread, mark read, save, batch actions). | Heavy RSS users; solves “too much clicking—need batch triage”. | `clawhub install rssaurus` |
| vercel | Manage Vercel from the command line: deploy, view logs, edit env vars, list projects, etc. | Web teams shipping on Vercel; solves “repetitive deploy/ops tasks”. | `clawhub install vercel` |
| wordpress | Automate WordPress via API: list posts, create drafts, update pages, manage categories/tags. | Content sites and ops teams; solves “publishing & site edits are repetitive”. | `clawhub install wordpress` |
| wpclaw-lite | For WooCommerce stores: pull orders/products via a connector for lightweight automation. | E-commerce operators; solves “need programmatic access to orders/products”. | `clawhub install wpclaw-lite` |
| pinch-to-post | A more full-featured WordPress automation toolkit: posts, media, SEO, bulk operations. | Teams building a content pipeline; solves “publishing + media + SEO are split across tools”. | `clawhub install pinch-to-post` |
| postiz | Multi-channel content scheduling: plan, queue, and publish across many platforms. | Growth/marketing ops; solves “posting everywhere is manual and hard to manage”. | `clawhub install postiz` |
| news-summary | Pulls news and generates readable summaries / key bullet points for a daily brief. | Anyone who wants the highlights fast; solves “too much news, not enough time”. | `clawhub install news-summary` |
| blogwatcher | Monitors blogs/RSS and detects updates automatically (“what changed since last time”). | Competitive/industry monitoring; solves “manually checking sites wastes time”. | `clawhub install blogwatcher` |
| clawops | Workflow orchestration: helps coordinate multiple skills into automated pipelines. | People running multiple automations; solves “skills are useful but not connected”. | `clawhub install clawops` |
| openclaw-anything | OpenClaw “toolbox” for common admin tasks: install, diagnose, connect channels, manage jobs. | OpenClaw admins/tinkerers; solves “too many scattered commands to remember”. | `clawhub install openclaw-anything` |
| openclaw-server-secure-skill | Server hardening checklist/scripts for machines running OpenClaw. | Production deployments; solves “server is exposed / needs baseline security”. | `clawhub install openclaw-server-secure-skill` |
| rss-ai-reader | Reads RSS → generates LLM summaries → pushes to Telegram/Feishu/Email. | Turning subscriptions into a deliverable “content product”; solves “too many feeds to read manually”. | `clawhub install rss-ai-reader` |
